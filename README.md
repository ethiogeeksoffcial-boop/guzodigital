## About

**Guzo Digital** is a digital marketing agency based in **Harar, Ethiopia**. We help businesses grow their online presence through strategic marketing, refined design, and measurable results.

This repository contains the source code for the official Guzo Digital website.

## ✨ Features

- **Hero Section** — Bold intro with call-to-action
- **About** — Agency story and stats
- **Services** — Social Media, Branding, Ads, Web Design, Content, Strategy
- **Portfolio** — Selected client work
- **Testimonials** — Real client feedback
- **Pricing** — Transparent ETB-based packages
- **Contact Form** — Book a call or send a message
- Fully **responsive** — works on mobile, tablet, and desktop

## 📦 How to Use

1. **Clone the repo**
   ```bash
   git clone https://github.com/ethiogeeksoffcial-boop/guzodigital.git
   ```

2. **Open in browser**
   ```bash
   cd guzodigital
   open index.html
   ```

3. **Edit content** — Update your real contact details, pricing, and portfolio in `index.html`
